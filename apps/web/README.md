# 💎 LuminaBI | Ultra-Premium Analytics Dashboard

LuminaBI is a state-of-the-art Business Intelligence platform designed for high-fidelity data visualization and seamless analytical workflows. Built with Next.js 15, Tailwind CSS 4, and ECharts, it provides a surgical-grade interface for data exploration.

## ✨ Key Features

-   **Dynamic Canvas**: Interactive, draggable, and resizable dashboard tiles.
-   **Intelligent Ingestion**: Automatic schema detection for CSV and Excel files.
-   **AI-Powered Insights**: Context-aware chart generation and data transformations.
-   **Premium Aesthetics**: Obsidian-themed dark mode with glassmorphic effects and fluid animations.
-   **Enterprise Ready**: Built with TypeScript, tRPC, and Prisma for a robust full-stack experience.

## 🚀 Quick Start

### Prerequisites

-   Node.js 20+
-   pnpm / npm / yarn

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

## 🛠 Tech Stack

-   **Framework**: [Next.js 15 (App Router)](https://nextjs.org)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
-   **Charts**: [Apache ECharts](https://echarts.apache.org)
-   **State Management**: [Zustand](https://zustand-demo.pmnd.rs)
-   **Animations**: [Framer Motion](https://framer.com/motion)
-   **Authentication**: [Clerk](https://clerk.com)
